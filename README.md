**WhatsApp Blast Application** — Go
Messaging automation tool built with the `whatsmeow` library, including anti-detection mechanisms, proxy rotation, and reconnect/retry handling.

**Multi-Tenant School CMS (SaaS)** — Laravel 12
Single-database, multi-tenant content management system for schools, using a hybrid architecture of static site generation with dynamic API-driven features (search, forms, visit counters).

**Pharmacy & Hospital POS Systems** — Go, Flutter, Svelte
Point-of-sale and inventory systems for pharmacy and hospital operations, covering procurement, inter-ward stock transfers, and prescription dispensing workflows.

Double-entry bookkeeping platform for Indonesian village-owned enterprises, including tax management (PPN, PPh), financial statements, and audit logging.
